StudentDatabase : Тестове завдання

Дане завдання повинно в результаті містити працюючу програму, яка задовольняє
всі нижчеописані вимоги, і якісний вихідний код.

Намагайтесь створити грамотну композицію класів і форматувати код відповідно до стандартів Java.

Опис завдання:

Реалізувати систему введення та відображення інформації про студентів інституту, що включає
наступні сутності та їх атрибути:

Student:
    first_name,
    last_name,
    date_of_birth,
    email,
    group,
    country

Система повинна мати таці фкнкції:

Відображення списку груп та студентів групи
Відображення списку студентів та фвльтрація за першою літерою призвища
Фільтрування списку студентів за призвищем та за номером групи
Додавання нового студента, реагування та видалення існуючого

ui (console) --> BL (service) --> DB (JDBC)
